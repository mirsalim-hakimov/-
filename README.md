# Random Task Generator

Автор: Хакимов Мирсалим

## Описание
GUI-приложение для генерации случайных задач.

## Установка и запуск

Клонируйте репозиторий:

git clone https://github.com/mirsalim-hakimov/random-task-generator.git

Перейдите в папку:

cd random-task-generator

Запустите:

python main.py

## Использование

1. Выберите категорию
2. Нажмите «Сгенерировать задачу»
3. Результат появится на экране

## Примеры тестов

Тест 1: Нажать «Сгенерировать задачу» → появляется случайная задача
Тест 2: Оставить поле пустым и нажать «Добавить» → ошибка
Тест 3: Выбрать «Спорт» → генерируются только спортивные задачи
